# DiffViewer: A simple viewer for Diff/AST

## Requirements

* [Node.js](https://nodejs.org/en/download/)

## Build

    $ npm install
    $ npm run build

## Usage (macOS)

The following assumes that Diff/AST is installed.

    $ diffast [--cache PATH] PATH0 PATH1
    $ ./run.py [--cache PATH] PATH0 PATH1
